DomShort

軽量でシンプルなJavaScript DOM操作ライブラリ - 依存関係なし、クールで高速

DomShortとは？

DomShortは、vanilla JavaScriptでDOM操作を簡単にする軽量ライブラリです。依存関係がなく、documentオブジェクトのメソッドを短く直感的に使えるラッパーを提供します。モダンブラウザ（Chrome、Firefoxなど）で動作し、小規模プロジェクトや高速なDOM操作に最適です。

例: d.id('myElement') で document.getElementById('myElement') を簡単に呼び出せます。LLM生成との互換性が高く、Grokのサポートで作成されました。

インストール

直接ダウンロード

domshort.js をダウンロードしてプロジェクトに含めてください：

<script type="module" src="path/to/domshort.js"></script>

使い方

ライブラリをインポートし、短いエイリアス（例: d）を使ってください：

import d from './domshort.js';

// 例: 要素を作成・操作
const div = d.create('div');
div.textContent = 'Hello, World!';
div.className = 'greeting';
document.body.appendChild(div);

非モジュール環境では、domshort.js の最後に window.DomShort = DomShort; を追加してグローバルに公開。
